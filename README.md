# MuJoCo Task-Embedding Pick and Place

A learning project for implementing robotic pick-and-place tasks in MuJoCo,
with task embeddings for representing and learning across task variations.

## Goals

- Build a reproducible MuJoCo pick-and-place environment.
- Train and evaluate reinforcement-learning baselines.
- Represent task variations with learned or structured task embeddings.
- Study generalization across objects, goals, and task configurations.
